Users can create manual events for team activities, meetings, or reminders.

---

## How to Create

1. Hover over a date
2. Click the **+** button that appears
3. Fill in:
   - **Title** — Event name
   - **Description** (optional)
   - **Visibility**:
     - **My Department** — Only your department sees it
     - **Public** — Everyone sees it
4. Click **Create**

---

## Editing and Deleting

- Click any event to view details
- If you created it, you can edit or delete it

---

## Limitations

<Callout type="info">
  For time-sensitive team communication, use a Notice via the Pulse. Calendar events are informational and do not trigger any notifications.
</Callout>

Calendar events are for reference only — they do not create Pulse notifications.
