The **Messaging** module is for conversations between team members.

---

## What Messaging Is

Messaging is for:
- Direct messages (one-to-one)
- Group conversations
- Discussing work

It is NOT for:
- Time-sensitive announcements → use Notices via Pulse
- Formal records → use the audit log

---

## What You Can Reference

The unique feature of Messaging is **record references**:

- Reference any SOP
- Reference any equipment
- Reference any Change Control

References appear as rich cards in the conversation. Recipients can click to open the record directly.
