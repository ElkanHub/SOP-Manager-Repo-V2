The **Reports** module provides historical data views. Unlike the live dashboard, reports show what happened in the past.

---

## Navigation

The Reports module features a modernized **horizontal tab system** running across the top of the screen. This design replaces legacy sidebars to give you maximum horizontal space for reading complex data tables and insights. Click on any tab to instantly switch between different reports.

---

## Report Types

1. **SOP Change History** — All SOP version changes
2. **Acknowledgement Log** — Who acknowledged what
3. **PM Completion Log** — All maintenance records
4. **Notice Log** — All notices sent (Admin only)
5. **AI Risk Insights** — AI-generated risk analysis (QA/Admin only)

---

## Common Features

- **Date range filter** — Select a time period
- **CSV export** — Download data for external analysis
