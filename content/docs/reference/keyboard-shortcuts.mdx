## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| <KeyboardShortcut keys={["Cmd", "K"]} /> | Focus search |
| <KeyboardShortcut keys={["Enter"]} /> | Send message |
| <KeyboardShortcut keys={["Shift", "Enter"]} /> | New line in message |

---

## Mac vs Windows

- On Mac, <KeyboardShortcut keys={["Cmd"]} /> shows as ⌘
- On Windows, it shows as Ctrl
