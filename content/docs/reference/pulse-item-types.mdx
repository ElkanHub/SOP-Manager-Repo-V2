## Item Types

| Type | Description | Group |
|------|-------------|-------|
| approval_request | SOP submitted for QA review | Priority |
| approval_update | QA responded to submission | Today |
| cc_signature | Change Control needs your signature | Priority |
| cc_deadline | Change Control deadline passed | Priority |
| pm_due | PM task due today | Today |
| pm_overdue | PM task is overdue | Priority |
| sop_active | New SOP version is active | Today |
| notice | Broadcast announcement | Messages |
| message | New chat message | Messages |
| todo | Personal to-do | Personal |
| system | Platform announcement | Varies |
